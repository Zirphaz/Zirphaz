# Residencial Matilde Carreiro Rental Website

Static website created for a small apartment rental business in Juazeiro do Norte, focused on direct local service, available unit discovery, and visit scheduling.

## Objective

Create a clear public presentation for apartment rentals, separating available and rented units while guiding interested visitors toward WhatsApp contact or a scheduled visit.

The communication rule of the project is strict: the site presents apartment rental only. It avoids house sale, property sale, buying, brokerage, or unrelated real-estate promises.

## Advertising and Propaganda Work

- Local positioning for a neighborhood rental business.
- Clear service promise: direct apartment rental with simple contact.
- Audience-first content for people comparing available units before calling.
- Copywriting for hero, calls to action, rental process, FAQ, and form feedback.
- Trust cues around location, availability, visit scheduling, and confirmation with the team.
- Visual direction with regional warmth, local references, custom imagery, and readable typography.
- Lead flow organized around available units, WhatsApp contact, and visit scheduling.

## Programming Work

- Responsive HTML structure for a commercial landing page.
- CSS system using local fonts, custom assets, hero layout, cards, filters, and mobile behavior.
- JavaScript property data model with available/rented status.
- Search, filter, sorting, metrics, property select, form feedback, and WhatsApp URL generation.
- Basic browser security posture with Content Security Policy and referrer policy.
- Vite development workflow with Prettier, Stylelint, JavaScript syntax check, and Playwright visual tests.
