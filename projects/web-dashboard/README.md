# Web Dashboard com JavaScript e React

Dashboard criado para apresentar uma visao simples de vendas, estoque e andamento de projetos. A ideia e mostrar organizacao de componentes, manipulacao de dados e construcao de interface com React.

## Funcionalidades

- Cards de metricas principais.
- Tabela de receita por categoria.
- Lista de projetos com status e progresso.
- Dados calculados com JavaScript.
- Layout responsivo.

## Tecnologias

- JavaScript
- React
- Vite
- CSS

## Como rodar

```bash
npm install
npm run dev
```

## Status

Primeira versao funcional. Pode evoluir depois para consumir API, integrar dados reais do sistema Python e adicionar graficos.
